## GATK reference data download
*know_dbsnp_vcf(Homo_sapiens_assembly38.dbsnp138.vcf, Homo_sapiens_assembly38.dbsnp138.vcf.idx) and interval_list(wgs_calling_regions.hg38.interval_list) files can be download in the following link:
https://console.cloud.google.com/storage/browser/genomics-public-data/resources/broad/hg38/v0

*gatk_germline_resource(af-only-gnomad.hg38.vcf.gz, af-only-gnomad.hg38.vcf.gz.tbi), gatk_panel_of_normal(1000g_pon.hg38.vcf.gz,1000g_pon.hg38.vcf.gz.tbi) and exac_common_knownsite (small_exac_common_3.hg38.vcf.gz, small_exac_common_3.hg38.vcf.gz.tbi) files can be download in the following link:
https://console.cloud.google.com/storage/browser/gatk-best-practices/somatic-hg38

Note: download reference with corresponding index files together and save all of the GATK reference data in one folder.
